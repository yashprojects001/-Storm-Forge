# 🌩️ Overview

**Thunder Temple Control Center** is a cinematic **advanced interactive frontend project** built as the command system of a floating storm civilization.

This is not a normal website.

This is a **living control system** where users command weather, route clouds, harvest lightning, protect floating islands, and power a celestial forge.
